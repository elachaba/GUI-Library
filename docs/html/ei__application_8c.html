◆ ei_app_create()

void ei_app_create ( ei_size_t  main_window_size,
bool  fullscreen 
)

Creates an application.

  • initializes the hardware (calls hw_init),
  • registers all classes of widget,
  • creates the root window (either in a system window, or the entire screen),
  • creates the root widget to access the root window.
Parameters
main_window_sizeIf "fullscreen is false, the size of the root window of the application. If "fullscreen" is true, the current monitor resolution is used as the size of the root window. hw_surface_get_size can be used with ei_app_root_surface to get the size.
fullScreenIf true, the root window is the entire screen. Otherwise, it is a system window.
